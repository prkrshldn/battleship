## Team Echo's Battleship / our capstone project

Created by @prkrshldn, @DavidShableski, and @vanselme1

This is an online Battleship game we made for our Project Implementation class in our senior year of college. 

The app was created using Vite.js, Socket.IO, and Supabase for Authentication.

You can sign up/log in and play against a CPU or another online player in a local friendly match or in a competitive match.

This app was created in 3 Sprints and is not perfect by any means, but it was a testing ground for our abilities in the real world.
